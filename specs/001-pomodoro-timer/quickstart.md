# Quickstart: 基本番茄鐘

**Branch**: `001-pomodoro-timer` | **Date**: 2026-02-17

## 環境需求

- Python 3.10 以上
- Tkinter（Python 標準安裝已包含）
- 無需安裝任何第三方套件

## 啟動應用程式

```bash
python src/main.py
```

## 基本操作

1. **開始工作**：點擊「開始」按鈕，25 分鐘倒數開始
2. **暫停計時**：點擊「暫停」按鈕，計時器停止
3. **繼續計時**：暫停後點擊「繼續」按鈕，從暫停處繼續
4. **重置計時**：點擊「重置」按鈕，回到工作模式 25:00
5. **工作完成**：倒數結束後會響起提示音，可選擇進入休息模式
6. **休息結束**：5 分鐘休息結束後響起提示音，自動回到工作模式

## 驗證功能正常

1. 啟動應用程式 → 看到 25:00 計時器與「開始」按鈕
2. 點擊開始 → 計時器開始倒數
3. 點擊暫停 → 計時器停止，按鈕變為「繼續」
4. 點擊繼續 → 計時器恢復倒數
5. 點擊重置 → 計時器回到 25:00
